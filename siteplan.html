Bilingual Learning Hub

Site Name

Project Name: Bilingual Learning Hub

Bilingual Learning Hub : This name reflects the mission to support learners in both English and French. It emphasizes connection, growth, and accessibility across cultures.

Site Purpose

The Bilingual Learning Hub is designed to provide a comprehensive, accessible, and engaging platform for language learners at all levels. This website is a bilingual learning resource designed to help users improve their English and French skills. It will include vocabulary, grammar, pronunciation, cultural tips, and interactive tools using modern web technologies. The website will offer:

  • English Learning Resources: Vocabulary lists, grammar fundamentals, and interactive practice exercises to help learners strengthen their English language skills.
  • French Learning Resources: Common phrases, pronunciation guides, cultural expressions, and context-based lessons to support learners in French-speaking environments.
  • Cultural Integration: Content that bridges language learning with cultural understanding, emphasizing the connection between language and culture.
  • Interactive Features: Dynamic content powered by JSON data and browser APIs, including vocabulary quizzes, translation utilities, and cultural fact displays.
  • Accessibility and Usability: A responsive, keyboard-navigable design that serves learners on all devices and with varying accessibility needs.

Scenarios

The following scenarios represent typical questions and needs of our target audience:

Scenario 1: "I want to improve my English vocabulary for professional settings."
Scenario 2: "I need practical French phrases for daily life in a French-speaking country."
Scenario 3: "How can I understand the cultural context behind these expressions?"
Scenario 4: "I want to test my knowledge and track my progress."

Color Scheme

The color scheme for the Bilingual Learning Hub is derived from the Chamber of Commerce project CSS variables:

Primary Color

#003366 (Dark Blue)

Usage: Header, footer, main headings (h1, h2), buttons, active navigation links

Secondary Color

#E67E22 (Orange)

Usage: Subheadings (h3, h4), accent borders, hover states, links

Accent Color

#2ECC71 (Green)

Usage: Highlights, backgrounds for featured content, cards, section dividers

Background & Text

#FFFFFF (White) & #333333 (Dark Gray)

Usage: Main background, body text, ensuring high contrast and readability

Typography

Merriweather (Serif) : Headings
Roboto (Sans-Serif) : Body Text
Ubuntu Mono (Monospace) : Code & Data

Wireframe Description

  • Header: Contains the site logo and navigation menu. On mobile, navigation is collapsed into a hamburger menu; on desktop, it displays horizontally.
  • Hero Section: A prominent welcome area with a headline, tagline, and background image that sets the tone for bilingual learning.
  • Feature Cards: Three cards highlighting the main offerings (English Learning, French Learning, Interactive Exercises). On mobile, they stack vertically; on desktop, they display in a 3-column grid.
  • Call-to-Action: A section encouraging users to get started or explore resources.
  • Footer: Contains copyright information, links, and contact details.

Accessibility Considerations

  • Semantic HTML structure (header, nav, main, footer)
  • Color contrast ratios meeting WCAG AA standards
  • Test performance and SEO with Lighthouse
  • Responsive design for all device sizes
